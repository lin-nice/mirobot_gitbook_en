##Step 1. Installing the pen
&ensp;&ensp;&ensp;&ensp;(1) Install the pen-holder module and the pen.<br/>
&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;•Fix the pen holder to the end of the robotic arm.<br/>
&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;•Tighten the jackscrew with M1.5 hex key, and fix the pen holder on the end motor shaft of the robotic arm.<br/>
&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;•Insert the pen into the holder from top.<br/>
&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;•Tighten the nut of the holder.<br/>

##Step 2. Set the pen height
&ensp;&ensp;&ensp;&ensp;We need to set the height of the pen first before drawing. Please follow the below instruction: <br/>
&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;•Connect the power supply and USB cable of Mirobot, and press the switch on button on the base.<br/>
&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;•Open the Mirobot Studio software. Confirm there is "CONNECTED" in the upper left corner, switch to the DRAWING panel, then click "HOMING" to reset the robotic arm.<br/>
&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;•After the manipulator back to the homing position, click the Z- button to adjust the height of the pen tip until it just touches the paper. <br/>
&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;•When the pen is closer to the paper, you can set the Step to "1".<br/>
&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;•The value of Z axis shall between around -10 and 35. Be careful not to over-adjust the Z axis position lower than it need to be, or the pen tip would hit the desktop too hard, interrupt the pen from moving and even cause damages to the joints. <br/>
&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;•When the pen tip is just touches the paper, click Set Pen Height button to set this pen height for the DRAWING function.

##Step 3. Letting the manipulator draw
&ensp;&ensp;&ensp;&ensp;After setting the pen height, the right panel will be automatically switched from “Pen Set” to the “Pattern Selection" panel sector. <br/>
&ensp;&ensp;&ensp;&ensp;•To add a graph from existing files, you can either choose an example from the Examples in the upper right “Pattern Selection" panel sector, or insert a graph by click Insert button. The graph you have chosen should be shown in the center canvas. 
<center><img src="http://lin88zhang.gitee.io/image_en/5/5-8.png" width="600"> </center>
&ensp;&ensp;&ensp;&ensp;•To add a text graph, typing in the lower right panel and click Add button, and the text you have typed should be placed in the center canvas.
<center><img src="http://lin88zhang.gitee.io/image_en/5/5-9.png" width="600"> </center>
&ensp;&ensp;&ensp;&ensp;•To add a mouse drawing graph, click the Edit | Draw button to switch from the Edit mode to the Draw mode. And then you can use your mouse as a pen to draw on the center canvas.
<center><img src="http://lin88zhang.gitee.io/image_en/5/5-10.png" width="600"> </center>
&ensp;&ensp;&ensp;&ensp;•To adjust the size and position of the graph, ensure the Edit | Draw button is in the Edit mode  click the graph in the canvas and a resize frame would appear around the graph. Adjust the corners to resize the graph and drag the inner frame to change the position of the graph. <br/>
&ensp;&ensp;&ensp;&ensp;•To delete the unwanted graph on the canvas, ensure the Edit | Draw button is in the Edit mode  choose the unwanted graph on the canvas  click Delete button.<br/>
&ensp;&ensp;&ensp;&ensp;•To start over from a blank canvas, click New button to clear the canvas.<br/>
&ensp;&ensp;&ensp;&ensp;•To start the drawing, click Run button, and the manipulator will draw the graph(s) on the canvas. <br/>
